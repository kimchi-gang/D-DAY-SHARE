# 🧑 사용자 기능 정의서

---

## 1️⃣ 인증 / 계정 관리

### 🔐 소셜 로그인
- Google 로그인
- GitHub 로그인
- 최초 로그인 시 자동 회원가입 처리

### 👤 계정 관리
- 내 프로필 조회
- 닉네임 수정
- 로그아웃
- 회원 탈퇴

### 🔔 알림 기본 설정
- 전체 알림 ON/OFF
- 알림 채널 선택
  - FCM (Android 푸시)
  - Kakao 알림톡

---

## 2️⃣ 그룹 기능

### 👥 그룹 관리
- 그룹 생성
- 그룹 삭제 (OWNER 권한)
- 그룹 이름 수정
- 그룹 오너 변경

### 📩 그룹 참여
- 초대 코드 발급
- 초대 코드로 그룹 참여
- 그룹 탈퇴
- 멤버 강퇴 (OWNER 전용)

### 📋 멤버 관리
- 그룹 멤버 목록 조회
- 멤버 역할 확인 (OWNER / MEMBER)

---

## 3️⃣ 개인 일정 기능

### 📅 일정 관리
- 개인 일정 생성
- 일정 수정
- 일정 소프트 삭제
- 중요 일정 표시

### 📆 일정 조회
- 오늘 일정 조회
- 이번 주 일정 조회
- 중요 일정 필터
- 사용자별 일정 필터 (그룹 캘린더 내)

### ⏳ D-Day 기능
- D-Day 자동 계산 및 표시
- D-Day 시작 오프셋 설정
  - 예: 7일 전부터 알림
  - 0일 → 당일만 알림

---

## 4️⃣ 일정 알림 기능 (고급 설정)

### ⏰ 일정별 알림 설정
- 일정별 개별 알림 시간 설정 (시/분)
- 알림 활성화 / 비활성화

### 🔁 반복 설정
- 매일 반복
- 평일 반복
- 요일 직접 선택 (CUSTOM)

### 🌏 시간대 설정
- 사용자 시간대 기준 알림 발송

### 📜 알림 관리
- 내가 받은 알림 목록 조회
- 알림 클릭 시 해당 일정으로 이동
- 알림 읽음 처리 (확장 기능)

---

## 5️⃣ 그룹 일정 기능

### 📌 일정 공유
- 개인 일정 생성 후 그룹에 공유
- 여러 그룹에 일정 공유 가능

### 📊 그룹 캘린더
- 그룹 전체 일정 조회
- 멤버 이름 기준 필터 조회
- 중요 일정 강조 표시

---

## 6️⃣ 그룹 공지 기능 (단발성 알림)

### 📢 공지 작성
- 그룹 공지 작성
- 공지 수정 / 삭제 (작성자 또는 OWNER)

### 🚀 공지 알림 발송
- 공지 작성 시 그룹 멤버 전체에게 즉시 1회성 푸시 발송
- FCM 기반 알림 전달

### 📂 공지 조회
- 공지 목록 조회
- 공지 상세 보기
- 공지 읽음 여부 확인 (확장 기능)
